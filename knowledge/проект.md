# Проект: Містраль

Дата старту: 2025
Платформа: Apple M2, macOS
Мова: Python 3.12

## Архітектура

- Groq LPU (~500 tok/s) — основний бекенд, модель llama-3.3-70b-versatile
- Ollama local (mistral-ua) — офлайн резерв, M2 ~20 tok/s
- web_server.py — FastAPI чат для iPhone (http://192.168.0.109:8000)
- mistral_chat.py — термінальний чат
- mistral_api.py — програмний API (клас MistralML)

## Ключові рішення

- num_ctx=32768, num_predict=4096 в Modelfile для великого контексту
- .env зберігає GROQ_API_KEY (не комітити в git)
- knowledge/ + memory.json для знань без перебудови моделі
